# Berkay Erdoğan Portfolio

This project is a modern and responsive personal portfolio website
built to showcase my frontend development skills and projects.

## 🚀 Demo

Live Demo:  
https://berkaychi.github.io/berkay-erdogan-portfolio/

## ✨ Features

- Responsive design
- Scroll-based active navigation
- Back to top button
- Light / Dark mode toggle
- Fade text animation
- Mobile optimized layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Design
- DOM Manipulation

## ⚙️ Installation

1. Clone the repository
   ```bash
   git clone https://github.com/berkaychi/berkay-erdogan-portfolio.git
   ```
2. Navigate to the project folder
   ```bash
   cd berkay-erdogan-portfolio
   ```
3. Open index.html in your browser
